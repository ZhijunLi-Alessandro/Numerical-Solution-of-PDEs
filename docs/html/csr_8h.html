Detailed Description

Header file for CSR matrix operations and iterative solvers.

This header file declares the SparseCSR structure and functions for creating, freeing, decomposing, performing matrix-vector multiplication, and solving linear systems using iterative methods (Jacobi, Gauss-Seidel, Conjugate Gradient) for sparse matrices stored in Compressed Sparse Row (CSR) format.
