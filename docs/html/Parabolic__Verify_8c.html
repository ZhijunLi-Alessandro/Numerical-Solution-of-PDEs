Detailed Description

Verification example for analytic Bessel/Hankel-based reference solutions.

This program evaluates the analytic/semi-analytic reference solution built from Bessel/Hankel functions and verifies the discrete assembly by comparing assembled RHS and operator actions against the manufactured solution. It writes diagnostic CSVs suitable for visualization and error analysis.
