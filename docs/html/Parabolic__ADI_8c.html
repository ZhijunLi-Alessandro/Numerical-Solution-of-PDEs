Detailed Description

Example: solve the 2D parabolic PDE with Dirichlet boundary conditions using ADI.

This example demonstrates constructing the mesh, assembling ADI split operators and time-stepping a manufactured solution using an alternating direction implicit (ADI) method. Output CSVs are produced for visualization and verification; see ReadMe.md for usage notes and expected outputs.
