Detailed Description

Header file for utility functions to print vectors and CSR matrices.

This header file declares functions for printing dense vectors, integer vectors, and sparse matrices in Compressed Sparse Row (CSR) format. The printing functions allow for specifying the number of decimal places for floating-point values.
