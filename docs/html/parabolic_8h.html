Detailed Description

API for parabolic/heat equation matrix and RHS assembly helpers.

This header exposes typedefs and assembly routines used by explicit and ADI finite-difference solvers for the parabolic PDE examples in the repository.

typedef double(* parabolic_Dirichlet_boundary) (double x, double y, double t, int boundary_type)

Function type for Dirichlet boundary values used by parabolic assembly.

The callback receives (x,y,t) and an integer boundary_type which allows callers to distinguish between multiple boundary segments.

◆ parabolic_source_term

typedef double(* parabolic_source_term) (double x, double y, double t, double hx, double hy)

Function type for parabolic source terms f(x,y,t;hx,hy).

The source term receives the spatial coordinates, current time, and the local cell sizes hx and hy in case the source depends on the mesh.

This fills the provided array b with contributions from the source term and Dirichlet boundary values for the current time t and step length tau.

Assemble the right-hand side vector for a parabolic time step.

For interior (active) points this fills the contribution from the source term integrated over the time-step; for boundary points it fills the Dirichlet boundary value via the provided callback.
