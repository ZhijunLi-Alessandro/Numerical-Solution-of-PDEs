Detailed Description

Declarations for Bessel/Hankel-based helper functions used by analytical / semi-analytical plane solutions in tests and examples.

This header exposes routines used to evaluate complex Bessel/Hankel functions (implemented in src/math/bessel.c) and higher-level helpers that compute plane-wave-like solutions and cell averages used by the parabolic/heat examples.

Evaluate an analytic plane-like solution built from Hankel functions.

The function computes the real part of e^{i t} * H_0^{(2)}(sqrt(-i) * r), where H_0^{(2)} is the Hankel function of the second kind and r is the radial coordinate. The small-radius behaviour is regularized by forcing a minimum r value.

Parameters
rRadial coordinate (r >= 0). Small values are clamped for stability.
tTime-like phase parameter.
Returns
The real-valued plane solution at (r,t).

Evaluate an analytic plane-like solution built from Hankel functions.

Implementation notes:

  • The function protects against r==0 by clamping a tiny positive radius.
  • Uses sqrt(-i) = exp(-i*pi/4) to form the complex argument z = sqrt(-i)*r.
  • Evaluates H_0^{(2)}(z) and multiplies by exp(i t), returning the real part.
Parameters
rRadius (distance from origin). Small values are clamped for stability.
tPhase parameter (real-valued).
Returns
Real-valued plane-solution (Re{ e^{i t} H_0^{(2)}(sqrt(-i) r) }).
Examples
Parabolic_ADI.c, Parabolic_Explicit.c, and Parabolic_Verify.c.
