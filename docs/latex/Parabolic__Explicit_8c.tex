\doxysubsection{Detailed Description}
Example\+: solve the 2D parabolic PDE with Dirichlet boundary conditions using an explicit scheme. 

This example constructs the grid, assembles the explicit time-\/step operator, and advances a manufactured/reference solution forward in time. It writes CSV output for the solution and exact/reference fields for visualization. See {\ttfamily Read\+Me.\+md} for run instructions and expected output.
