\doxysubsection{Detailed Description}
Test program for 5x5 CSR matrix operations and iterative solvers. 

This test program creates a 5x5 sparse matrix in CSR format, defines a right-\/hand side vector, and solves the linear system using Jacobi, Gauss-\/\+Seidel, and Conjugate Gradient methods. The results are printed to the console.

Usage\+: Compile the program and run it.

Example\+: \begin{DoxyVerb}mkdir build && cd build
cmake ..
make
../bin/test_csr_5x5 \end{DoxyVerb}
