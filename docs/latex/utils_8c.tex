\doxysubsection{Detailed Description}
Utility functions for printing vectors and \doxylink{structSparseCSR}{Sparse\+CSR} matrices. 

This file contains functions to print dense vectors, integer vectors, and sparse matrices in Compressed Sparse Row (CSR) format. The printing functions allow for specifying the number of decimal places for floating-\/point values.

Functions\+:
\begin{DoxyItemize}
\item print\+\_\+vector\+: Print a dense vector with specified decimal places.
\item print\+\_\+int\+\_\+vector\+: Print an integer vector.
\item print\+\_\+\+Sparse\+CSR\+: Print a \doxylink{structSparseCSR}{Sparse\+CSR} matrix in dense format with specified decimal places.
\item print\+\_\+\+Sparse\+CSR\+\_\+simple\+: Print the internal representation of a \doxylink{structSparseCSR}{Sparse\+CSR} matrix.
\end{DoxyItemize}
