\doxysubsection{Detailed Description}
Structure to represent a sparse matrix in Compressed Sparse Row (CSR) format. 

CSR format stores a sparse matrix using three arrays\+:
\begin{DoxyItemize}
\item row\+\_\+ptr\+: Array of size (rows + 1) where row\+\_\+ptr\mbox{[}i\mbox{]} indicates the start of row i in col\+\_\+ind and values.
\item col\+\_\+ind\+: Array of size nnz (number of non-\/zero elements) storing the column indices of non-\/zero elements.
\item values\+: Array of size nnz storing the non-\/zero values of the matrix. 
\end{DoxyItemize}\begin{Desc}
