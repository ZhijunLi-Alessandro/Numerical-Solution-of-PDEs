\doxysubsection{Detailed Description}
Example program to solve a 2D Poisson problem with Neumann boundary condition. 

This program define a Poisson problem of Neumann boundary conditions on the two-\/ dimensional irregular computational region of periodic solutions and constructs a finite difference method to numerically solve and calculate it. The solution is set to be periodic.
