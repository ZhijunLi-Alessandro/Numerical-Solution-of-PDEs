\doxysubsection{Detailed Description}
Header file for a uniform grid structure and related functions. 

This header file declares the Grid structure and functions to create and free a grid. The Grid structure represents a 2D grid with specified dimensions and spacing. This file also declares functions for creating, initializing, and data remapping. \begin{DoxySeeAlso}{See also}
