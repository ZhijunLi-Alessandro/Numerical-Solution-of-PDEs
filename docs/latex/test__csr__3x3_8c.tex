\doxysubsection{Detailed Description}
Test program for 3x3 CSR matrix operations and iterative solvers. 

This test program creates a 3x3 sparse matrix in CSR format, defines a right-\/hand side vector, and solves the linear system using Jacobi, Gauss-\/\+Seidel, and Conjugate Gradient methods. The results are printed to the console.

Usage\+: Compile the program and run it.

Example\+: \begin{DoxyVerb}mkdir build && cd build
cmake ..
make
../bin/test_csr_3x3 \end{DoxyVerb}
